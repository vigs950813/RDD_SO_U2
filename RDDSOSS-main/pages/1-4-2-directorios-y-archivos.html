1.4.2 Directorios y archivos

Es un sistema de organización de archivos que permite al usuario acceder y localizar fácilmente los archivos y ofrece una visión lógica simple al usuario, muy diferente del modo de almacenamiento auténtico. Un directorio es un objeto o estructura de datos compuesto de entradas, cada una de las cuales contiene información sobre el archivo al que se refiere (MS-DOS) o un puntero y una estructura de datos que contiene dicha información.

El nombre de ruta de cada archivo dentro de la clasificación de directorios se puede obtener desde el directorio raíz, el punto más alto de la clasificación de directorios. Los nombres de ruta absolutos similares incluyen la lista de directorios que se deben pasar partiendo del para acceder al archivo, divida los componentes con diagonales.

Estructuras de archivos

Es una estructura de datos que se encuentra dentro de una unidad lógica que permite al sistema operativo almacenar datos de manera organizada e independiente de los procesos que los utilizan.

Las propiedades físicas de varios dispositivos son abstraídas por el sistema de archivos.

Tipos de estructuras de archivos

Tener un solo nivel de directorios en el sistema de archivos es el método más sencillo.

Permitir múltiples niveles de directorios es ideal. Esto se puede lograr mediante la inclusión de archivos de tipo directorio dentro de los directorios en los que se crea una la estructura jerárquica de directorios en forma de árbol se conoce como ruta absoluta de un archivo, que es el camino desde la raíz del archivo hasta el final.

Sería deseable tener caminos para fortalecer la estructura del árbol anterior, los archivos de tipo enlace simbólico, también conocidos como enlaces suaves, permiten el acceso directo a otros directorios y permiten que un archivo esté en más de un directorio.
