Glosario

  • Argumento

    Un valor que se pasa a la función para que la utilice en su procesamiento.

  • BSP (Paquete de Soporte para Placa)

    Software que contiene controladores específicos de hardware y otras rutinas que permiten que un sistema operativo en particular.

  • CLI (Interfaz de Línea de Comandos)

    Interfaz que permite a los usuarios interactuar con el sistema ingresando comandos de texto en lugar de utilizar una interfaz gráfica.

  • CPU (Unidad Central de Procesamiento)

    Componente de una computadora, se encarga de ejecutar instrucciones de programas, realizar cálculos y coordinar las operaciones de otros componentes para procesar datos y llevar a cabo tareas.

  • Declaración

    Línea de código que da instrucciones a la computadora para llevar a cabo una acción o definir una entidad, como variables, funciones o estructuras, en un programa.

  • Función

    Bloque de código que realiza una tarea específica y puede ser reutilizado llamándolo desde otros lugares de otro programa que puede ser de gran escala.

  • GUI (Interfaz Gráfica de Usuario)

    Software que permite a los usuarios interactuar con el sistema utilizando elementos visuales como ventanas, iconos y menús en lugar de comandos de texto.

  • Hardware

    Componentes físicos y electrónicos de una computadora u otro dispositivo, como la CPU, memoria, disco duro, pantalla y dispositivos de entrada/salida, que trabajan juntos para ejecutar programas y realizar tareas.

  • Interrupción

    Señal que detiene momentáneamente la ejecución de un programa para manejar eventos urgentes o solicitudes del hardware, permitiendo una respuesta rápida y eficiente a situaciones como errores, solicitudes de entrada/salida y otros eventos importantes.

  • Proceso

    Instancia en ejecución de un programa, con su propio espacio de memoria y recursos asignados, que realiza tareas específicas de manera independiente dentro del sistema computacional.

  • RAM (Memoria de Acceso Aleatorio)

    Componente físico de una computadora que permite el almacenamiento temporal de información para su procesamiento y uso.

  • RTOS (Sistema Operativo en Tiempo Real)

    Software diseñado para gestionar tareas y procesos de manera inmediata y predecible, garantizando que las respuestas se produzcan en un tiempo determinado y constante, esencial en aplicaciones donde la respuesta rápida es crítica, como en sistemas de control industrial o dispositivos médicos.

  • Script

    Conjunto de instrucciones cortas escritas en un lenguaje de programación para ejecutar tareas específicas.

  • Sistema operativo

    Software que administra recursos y facilita la interacción entre el hardware y otros programas en una computadora.

  • Software

    Conjunto de programas, instrucciones y datos que controlan el funcionamiento de una computadora o dispositivo, permitiendo realizar tareas y procesar información de manera programada.
