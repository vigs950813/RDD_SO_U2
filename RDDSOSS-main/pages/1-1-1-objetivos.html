1.1.1 Objetivos

El sistema operativo es el único de los programas que se encarga de interactuar con el hardware de la computadora. las funciones principales son:

  • Abstracción:

    Cada programa debe de tener acceso con el hardware y su configuración con los equipos de cómputo derivado a que el sistema operativo es el responsable de ofrecerle una lista de abstracciones a los desarrolladores y con ellos puedan cumplir las necesidades del usuario como puede ser que los archivos y directorios estén almacenados correctamente y a su vez organizados.

  • Administración de recursos:

    Se refiere a la memoria, espacio de almacenamiento, tiempo de procesamiento, acceso a los controladores de sus dispositivos, etc. Por ello los diferentes procesos que se lleven a cabo y se ejecuten debe de existir una orden para llevarlos a cabo; por ende, el sistema operativo gestiona toda la asignación de recursos en donde puede implementar políticas para que se asignen en forma efectiva, y dependiendo de las necesidades, los recursos del sistema, sus archivos y directorios como dispositivos de almacenamiento.

  • Aislamiento:

    En un sistema multiusuario y multitarea, cada proceso que genera el usuario no debe de preocupar que al mismo tiempo otro esté requiriendo recursos; ya que, el sistema implementa mecanismos en donde las funciones se ejecuten correctamente otorgando los recursos necesarios a cada uno. Esto dependerá de la importancia del proceso que cada uno esté ejecutando para el sistema operativo.

Características

Las características principales del sistema operativo son:

  • Facilidad de uso:

    El sistema operativo se encarga de facilitar el uso de la computadora.

  • Eficiencia:

    Un sistema operativo es aquel que permite que los recursos de una computadora se utilizar de una manera eficiente

  • Capacidad para evolucionar:

    Un sistema operativo se construye de tal forma que se pueden desarrollar e implementar nuevas funciones de operatividad del mismo sin interferir con sus procesos y servicios que tiene.

Por lo que el objetivo del sistema operativo es administrar, controlar y distribuir los recursos requeridos tanto por el usuario como por el sistema en una forma organizada y dependiendo de la prioridad del mismo.
