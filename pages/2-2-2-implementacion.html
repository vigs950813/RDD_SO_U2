2.2.2 Implementación

Cada Sistema operativo implementa los hilos del sistema utilizando tres estándares en donde las librerías nativas son:

  1. Win32.
  2. OS/2.
  3. POSIX.

Las dos primeras son propietarias y sólo corren bajo sus respectivas plataformas (NT, Win95, OS/2).


La especificación POSIX (IEEE 1003.1c) conocida como pthreads [Butenhof, 1997] está pensada para todas las plataformas y está disponible para la mayoría de las implementaciones UNIX y Linux, así como VMS y AS/400.


Algunas de las implementaciones del estándar POSIX son del primer tipo, mientras que tanto OS/2 como Win32 son del segundo tipo. Ambos métodos pueden ser usados para implementar exactamente el mismo API.


Existen dos formas de implementación de los hilos en el sistema operativo:

  1. Es escribir una librería al nivel de usuario en donde todas las estructuras y el código de la librería estarán en espacio de usuario, por lo que la mayoría de las llamadas que se hagan desde la librería se ejecutarán en el espacio de usuario y no hará más uso de las rutinas del sistema que cualquier otra librería o programa de usuario.
  2. La implementación al nivel de núcleo que es donde la mayoría de las llamadas de la librería requerirán llamadas al sistema.

                //Ejemplo 1:
                //En este modelo, varios hilos de usuario se mapean a un número menor de hilos del sistema operativo,
                // permitiendo una implementación más eficiente y adaptativa de hilos.   

                #include 
                #include 
 
                void *funcion_hilo(void *arg) {
	                printf("Hilo ejecutado: %d\n", *((int *) arg));
	                pthread_exit(NULL);
                }
 
                int main() {
	                pthread_t hilos[4];
	                int ids[] = {1, 2, 3, 4};
 
	                // Crear hilos
	                for (int i = 0; i < 4; i++) {
                        pthread_create(&hilos[i], NULL, funcion_hilo, (void *) &ids[i]);
	                }
 
	                // Esperar la finalización de los hilos
	                for (int i = 0; i < 4; i++) {
    	                pthread_join(hilos[i], NULL);
	                }
 
	                return 0;
                }
